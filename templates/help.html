Instructions

Suivi :


1) Entrez le nom de votre équipe, puis entrez dans le système.

2) Appuyez sur le bouton Démarrer pour entrer dans le labyrinthe.

3) Explorez le labyrinthe à la recherche d'indices.

4) Ne vous laissez pas vaincre par le système !

5) Pour quitter, trouvez la sortie ou appuyez sur le bouton Quitter.

Attaques du Système :


Pendant votre exploration, le système vous attaquera aléatoirement avec un type d'attaque.

À vous de contrer son attaque ou de contre-attaquer !

> Si vous contrez son attaque : rien ne se passera.
> Tandis que, si vous arrivez à gagner : un point bonus vous sera attribué !
> Et, évidemment, si le système vous vainc : vous perdrez un point.
